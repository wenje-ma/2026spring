## 3.5

### 4

设$X_1,X_2,\cdots,X_n$是来自两点分布总体$B(1,p)$的样本，其中$0<p<1$为未知参数。

（1）求$p^m$的一致最小方差无偏估计，其中$m>0$为自然数。

（2）求$p(1-p)$的一致最小方差无偏估计。

#### 解答

（1）

两点分布的充分完备统计量为$T = \sum X_i$。

取简单的无偏估计：$\hat{g} = \prod_{i=1}^n X_i$，

显然$E(\hat{g}) = \prod_{i=1}^n EX_i = p^n$。

计算条件期望$E(X_1 \cdots X_n | T = t)$：

- 当$t < n$时，$X_i$至少有一个为0，因此期望为0；
- 当$t \ge n$时，

$E(X_1 \cdots X_n | T = t) = P(X_1, \cdots, X_n = 1 | T = t) $

$= \frac{P(X_1=1, \cdots, X_n=1, \sum_{i=1}^n X_i = t)}{P(T = t)}$

$= \frac{\binom{t-n}{n} p^n (1-p)^{t-n}}{\binom{n}{t} p^t (1-p)^{n-t}}$

$= \frac{t(t-1)\cdots(t-n+1)}{n(n-1)\cdots(n-t+1)}$，

因此$p^n$的UMVUE为

$\frac{T(T-1)\cdots(T-n+1)}{n(n-1)\cdots(n-t+1)} I_{\{T \ge n\}}$，其中$T=\sum X_i$。

（2）

由于两点分布的方差为$p(1-p)$，

样本方差$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i-\bar{X})^2$

本身就是总体方差的无偏估计，

因此将$S^2$用$T$表示：已知$T = \sum X_i$，

对于两点分布，$X_i^2 = X_i \in \{0,1\}, \forall i$，

因此

$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i-\bar{X})^2$

$= \frac{1}{n-1}(\sum_{i=1}^n X_i^2 - n\bar{X}^2) $

$= \frac{1}{n-1}(\sum_{i=1}^n X_i - \frac{1}{n}(\sum X_i)^2)$

$= \frac{1}{n-1}(T - \frac{T^2}{n}) = \frac{T(n-T)}{n(n-1)}$。

### 6

设$X_1,X_2,\cdots,X_n$是来自正态总体$N(0,\sigma^2)$的样本，其中$\sigma>0$为未知参数，分别求$\sigma$和$\sigma^4$的一致最小方差无偏估计。

#### 解答

均值已知为0，则正态分布的充分统计量为$\sum X_i^2$。

已知正态分布的样本方差服从$\chi^2$分布：$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，

因此$T = \sum X_i^2 \sim \sigma^2 \chi^2(n)$。

求$\sigma$的UMVUE时，自然地考虑$\sqrt{T}$：

因为$\sqrt{T}$与$\sigma$大概率有线性关系，由$T \sim \sigma^2 \chi^2(n)$得知

$E(\sqrt{T}) = \sigma E(\sqrt{\chi^2(n)}) = \sigma E(\sqrt{\chi^2(n)})$，

$E(\sqrt{\chi^2(n)}) = \int_0^\infty \sqrt{t} \frac{t^{\frac{n}{2}-1} e^{-\frac{t}{2}}}{2^{\frac{n}{2}}\Gamma(\frac{n}{2})} dt = \sqrt{2} \frac{\Gamma(\frac{n+1}{2})}{\Gamma(\frac{n}{2})}$。

因此$E(\sqrt{T}) = \sigma \sqrt{2} \frac{\Gamma(\frac{n+1}{2})}{\Gamma(\frac{n}{2})}$，

因此$\frac{\Gamma(\frac{n}{2})}{\sqrt{2}\Gamma(\frac{n+1}{2})} \sqrt{\sum_{i=1}^n X_i^2}$为$\sigma$的UMVUE。

求$\sigma^2$的UMVUE时，自然地考虑$T^2$：

$E(T^2) = \sigma^4 E((\chi^2(n))^2)$，

已知$E(\chi^2(n))=n$，$\text{Var}(\chi^2(n))=2n$，

则$E((\chi^2(n))^2)=n(n+2)$，$E(T^2) = \sigma^4 n(n+2)$。

因此$\frac{1}{n(n+2)}(\sum_{i=1}^n X_i^2)^2$是$\sigma^4$的UMVUE。

### 7

设$X_1,X_2,\cdots,X_n$是来自正态总体$N(\mu,\sigma^2)$的样本，其中$\mu\in\mathbb{R},\ \sigma>0$都是未知参数。

（1）分别求$\mu+\sigma^2$和$\mu^2/\sigma^2$的一致最小方差无偏估计；  

（2）证明$\hat{\mu}^2=\bar{X}^2-\frac{1}{n}S^2$是$\mu^2$的一致最小方差无偏估计。

#### 解答

（1）

已知$\bar{X}$是$\mu$的UMVUE，$S^2$是$\sigma^2$的UMVUE。

因此$\mu+\sigma^2$的UMVUE为$\bar{X} + S^2$。

由于$\bar{X}$与$S^2$独立，因此自然考虑$E(\frac{T_1}{T_2})$，其中$T_1=\bar{X}, T_2=\sum_{i=1}^n (X_i-\bar{X})^2$。

$E(\frac{T_1^2}{T_2}) = E(T_1^2)E(\frac{1}{T_2})$。

已知$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i \sim N(\mu, \frac{\sigma^2}{n})$，

则$E(\bar{X}^2) = D(\bar{X}) + (E\bar{X})^2 = \frac{\sigma^2}{n} + \mu^2$。

已知$T_2 \sim \sigma^2 \chi^2(n-1)$，

因此$E(\frac{1}{T_2}) = \frac{1}{\sigma^2(n-3)}$，

即$E(\frac{\bar{X}^2}{T_2}) = \frac{1}{n(n-3)} + \frac{\mu^2}{\sigma^2(n-3)}$。

因此$(n-3)\frac{\bar{X}^2}{\sum_{i=1}^n (X_i-\bar{X})^2} - \frac{1}{n}$为$\frac{\mu^2}{\sigma^2}$的UMVUE。

（2）

无偏性：

$E(\bar{X}^2 - \frac{1}{n}S^2) = E\bar{X}^2 - \frac{1}{n}E(S^2) = \mu^2 + \frac{\sigma^2}{n} - \frac{\sigma^2}{n} = \mu^2$，

故确是无偏估计。

充分完备性：

$\bar{X}^2 - \frac{1}{n}S^2$是$(\bar{X}, S^2)$的函数，的确具有充分完备性，

依据L-S定理，$\hat{\mu}^2$的确是$\mu^2$的UMVUE。
