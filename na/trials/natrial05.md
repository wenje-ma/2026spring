## 1

试应用北太天元软件编写脚本函数实现Newton插值，要求能输出插值多项式。对函数$f(x)=\frac{1}{1+4x^2}$在区间$[-5,5]$上实现10次多项式插值。

（1）输出插值多项式；

（2）在区间$[-5,5]$内均匀插入99个节点，计算这些节点上函数$f(x)$的近似值，并在同一张图中画出原函数和插值多项式的图形；

（3）观察Runge现象，计算插值函数在各节点上的误差，并画出误差图。

### 解答

（1）

求10次多项式，需要取11个等距点，代入函数文件中求解差值多项式：

```matlab
clear;clc;
syms x;
f(x)=1/(1+4*x^2);
n_interp=10;
X=linspace(-5,5,n_interp+1);
Y=double(f(X));
L=nanewtoninterp(X,Y);
pretty(L);
```

```matlab
function f = nanewtoninterp(X,Y)
n = length(X);
D = zeros(n);
D(:,1) = Y(:);
for j = 2:n
    for i = j:n
        D(i,j) = (D(i,j-1)-D(i-1,j-1))/(X(i)-X(i-j+1));
    end
end
C = diag(D);
syms x; f = C(1);
p = 1;
for k = 2:n
    p = p*(x-X(k-1));
    f = f + C(k)*p;
end
f = expand(f);
end
```

```matlab
clear; clc; close all;
syms x;
f(x) = 1/(1 + 4*x^2);
n_interp = 10;
X = linspace(-5, 5, n_interp+1);
Y = double(f(X));
L = nanewtoninterp(X, Y);
pretty(L);
x_plot = linspace(-5, 5, 99);  
f_true = double(f(x_plot));
f_interp = double( subs(L, x, x_plot) );
figure(1);
plot(x_plot, f_true);
hold on;
plot(x_plot, f_interp);
scatter(X, Y, 50, 'k', 'filled', 'DisplayName', '插值节点');
xlabel('x'); ylabel('y');
legend; grid on;
```
