##3

已知函数表如下：

|$x_i$|1|3|4|6|
|------------|----|----|----|----|
|$f(x_i)$|-7|5|8|14|

试求其三阶Lagrange插值多项式，并以此计算$ f(2) $的近似值。

### 解答

$n$阶Lagrange插值多项式为：$L_n(x) = \sum_{i=0}^n y_i l_i(x)$，

其中$l_i(x) = \prod_{\substack{j=0 \\ j \neq i}}^n \frac{x - x_j}{x_i - x_j}$。

因此：

$l_0(x) = \frac{(x-3)(x-4)(x-6)}{(1-3)(1-4)(1-6)} = -\frac{1}{30}(x-3)(x-4)(x-6)$，

$l_1(x) = \frac{(x-1)(x-4)(x-6)}{(3-1)(3-4)(3-6)} = \frac{1}{6}(x-1)(x-4)(x-6)$，

$l_2(x) = \frac{(x-1)(x-3)(x-6)}{(4-1)(4-3)(4-6)} = -\frac{1}{6}(x-1)(x-3)(x-6)$，

$l_3(x) = \frac{(x-1)(x-3)(x-4)}{(6-1)(6-3)(6-4)} = \frac{1}{30}(x-1)(x-3)(x-4)$，

$L_3(x) = 7 \cdot (-\frac{1}{30})(x-3)(x-4)(x-6) + 5 \cdot \frac{1}{6}(x-1)(x-4)(x-6)$，

$+ 8 \cdot (-\frac{1}{6})(x-1)(x-3)(x-6) + 14 \cdot \frac{1}{30}(x-1)(x-3)(x-4)$，

$= \frac{1}{5}x^3 - \frac{8}{5}x^2 + \frac{49}{5}x - \frac{92}{5}$，

$L_3（2） = 0.4$。

##7

设$x_0,x_1,\dots,x_n$为$ n+1 $个互异节点，$l_j(x)(j=0,1,\dots,n)$为这组节点上的$ n $次Lagrange插值基函数，试证：

（1）$\sum_{j=0}^n(x_j-x)^kl_j(x)\equiv0\quad(k=1,2,\dots,n)$；

（2）设$ P(x) $是任意一个最高次项系数为1的$ n+1 $次多项式，则

$$
P(x)-\sum_{j=0}^nP(x_j)l_j(x)=\omega_{n+1}(x)=\prod_{j=0}^n(x-x_j)
$$

### 解答

（1）

展开二项式：$(y-x)^k = \sum_{j=0}^k \binom{k}{j} (-1)^{k-j} x^j y^{k-j}$，

代入求和：$\sum_{i=0}^n (x_i - x)^k l_i(x) = \sum_{j=0}^k \binom{k}{j} (-1)^{k-j} x^j \sum_{i=0}^n x_i^{k-j} l_i(x)$。

已知对任意次数≤n的多项式$f(x)$，

有$\sum_{i=0}^n f(x_i) l_i(x) \equiv f(x)$，

此时$f(x) = x^{k-j}$，$k=1,2,\dots,n$，$i=1,2,\dots,k$，

因此$k-i \leq n$，即$\sum_{i=0}^n x_i^{k-j} l_i(x) = x^{k-j}$。

因此$\sum_{i=0}^n (x_i - x)^k l_i(x) = \sum_{j=0}^k \binom{k}{j} (-1)^{k-j} x^j x^{k-j} = (x_i - x)^k = 0$。

（2）

$\sum_{i=0}^n P(x_i) l_i(x)$是$P(x)$在节点$x_0,\dots,x_n$上的n次Lagrange插值多项式，

因此次数$≤n$。

$P(x)$是首1的n+1次多项式，

因此$P(x) - \sum_{i=0}^n P(x_i) l_i(x)$也是首1的n+1次多项式。

当$x=x_0,\dots,x_n$时，

$P(x_k) - \sum_{i=0}^n P(x_i) l_i(x_k) = P(x_k) - P(x_k) = 0$。

因此一个以$x_0,\dots,x_n$为零点的n+1次多项式，

只能为$\omega_{n+1}(x) = \prod_{j=0}^n (x - x_j)$。

##8

已知函数表如下：

|------------|--------|--------|--------|--------|
|$f(x_i)$|2.41450|2.46259|2.65271|3.03035|

试求其三阶Newton插值多项式，并以此计算$ f(1.682) $的近似值。

### 解答

差商表：

$f[x_0,x_1] = \frac{f_1 - f_0}{x_1 - x_0} \approx 2.531053$，

$f[x_1,x_2] = \frac{f_2 - f_1}{x_2 - x_1} \approx 2.795882$，

$f[x_2,x_3] = \frac{f_3 - f_2}{x_3 - x_2} \approx 2.997143$，

$f[x_0,x_1,x_2] = \frac{f[x_1,x_2] - f[x_0,x_1]}{x_2 - x_0} \approx 3.044011$，

$f[x_1,x_2,x_3] = \frac{f[x_2,x_3] - f[x_1,x_2]}{x_3 - x_1} \approx 1.037428$，

$f[x_0,x_1,x_2,x_3] = \frac{f[x_1,x_2,x_3] - f[x_0,x_1,x_2]}{x_3 - x_0} \approx -9.420577$，

$N_3(x) = f_0 + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1)$

$+ f[x_0,x_1,x_2,x_3](x-x_0)(x-x_1)(x-x_2)$

$= 2.331053x - 1.673150595$，

则$N_3(1.682) = 2.58408$。

##10

已知函数表如下：
|$x_i$|1|2|3|4|
|------------|----|----|----|----|
|$f(x_i)$|3|5|9|15|

分别用Newton向前、向后插值公式计算$f(1.5),f(3.7)$的近似值。

### 解答

向前插值：

$\Delta f_0 = f_1 - f_0 = 2$，

$\Delta f_1 = f_2 - f_1 = 4$，

$\Delta f_2 = f_3 - f_2 = 6$，

$\Delta^2 f_0 = \Delta f_1 - \Delta f_0 = 2$，

$\Delta^2 f_1 = \Delta f_2 - \Delta f_1 = 2$，

$\Delta^3 f_0 = \Delta^2 f_1 - \Delta^2 f_0 = 0$，

$f(x_0 + th) = \sum_{k=0}^n \binom{t}{k} \Delta^k f_0$，

本题为$f(1+t) = t^2 + t + 3$。

因此$f(1.5) = 3.75$，$f(3.7) = 12.99$。

向后插值：

$\nabla f_1 = f_1 - f_0 = 2$，

$\nabla f_2 = f_2 - f_1 = 4$，

$\nabla f_3 = f_3 - f_2 = 6$，

$\nabla^2 f_2 = \nabla f_3 - \nabla f_2 = 2$，

$\nabla^2 f_3 = \nabla f_3 - \nabla f_2 = 2$，

$\nabla^3 f_3 = \nabla^2 f_3 - \nabla^2 f_2 = 0$，

$f(x_n + th) = \sum_{k=0}^n \binom{t + k - 1}{k} \nabla^k f_n$，

在本题中，$f(4 + t) = t^2 + 5t + 15$，

因此$f(1.5) = 8.75$，$f(3.7) = 13.59$。

##12

在$[0,\pi/2]$上给出$f(x)=\sin x$的等距节点函数表，为保证用分段线性插值求非节点处的正弦值时，其截断误差不超过$ 10^{-6} $，函数表的步长至少应取多大？若改用分段二次插值求非节点处的正弦值，步长应取多大？

### 解答

由Lagrange余项公式：

$R_1(x) = f(x) - L_1(x) = \frac{f''(\xi)}{2!} (x - x_0)(x - x_1)$，

取$M = \max |f''(x)|$，令$x - x_i = t$，步长为$h$，则$t(t-h) \leq \frac{h^2}{4}$，

则$|R_1(x)| \leq \frac{h^2}{8} M$。

本题中，$M = \max |\sin x| = 1$，$\frac{h^2}{8} \leq 10^{-6}$，

则$h \leq 2.828 \times 10^{-3}$，至少取$2.828 \times 10^{-3}$。

由Lagrange余项公式：

$|R_2(x)| = |f(x) - L_2(x)| = \frac{f^{（3）}(\xi)}{3!} (x - x_0)(x - x_1)(x - x_2)$，

令$M_3 = \max_{x_0 \leq x \leq x_2} |f^{（3）}(x)|$，

令$x - x_i = t$，步长为$h$，

则$|t(t-h)(t-2h)| \leq \frac{2h^3}{3\sqrt{3}}$，

则$|R_2(x)| \leq \frac{h^3}{9\sqrt{3}} M_3$。

本题中，$M_3 = \max |\sin x| = 1$，$\frac{h^3}{9\sqrt{3}} \leq 10^{-4}$，

则$h \leq 0.02498$，至少取$0.02498$。

##16

给定函数表

|$x_i$|0|1|2|
|-------------|----|----|----|
|$f(x_i)$|1|2|1|
|$f'(x_i)$||0|-1|

且已知$f(x)$在$[0,2]$上五阶连续可导，求$f(x)$的四次Hermite插值多项式$H_4(x)$，并写出余项$f(x)-H_4(x)$的表示式。

### 解答

设$H_4(x) = a_0 + a_1x + a_2x^2 + a_3x^3 + a_4x^4$。

代入表格数据：

$\begin{cases}a_0 = 1 \\a_1 + a_2 + a_3 + a_4 = 2 \\1 + 2a_1 + 4a_2 + 8a_3 + 16a_4 = 1 \\4a_4 + 3a_3 + 2a_2 + a_1 = 0 \\32a_4 + 12a_3 + 4a_2 + a_1 = -1\end{cases}$

解得：

$\begin{cases}a_0 = 1 \\a_1 = 4 \\a_2 = -6.75 \\a_3 = 0.5 \\a_4 = 0.25\end{cases}$

因此$H_4(x) = 0.25x^4 + 0.5x^3 - 1.75x^2 + 4x + 1$。

本题为整段4次Hermit，因此需3个条件，分项导数为5阶。

$f(x) - H_4(x) = \frac{f^{(5)}(\xi)}{5!} x(x-1)^2(x-2)^2$。

##17

已知函数表如下：

|$x_i$|0|1|2|3|
|---------|----|----|----|----|
|$y_i$|0|0|0|0|

边界条件为

（1）$m_0=1,m_3=0$；
（2）$M_0=1,M_3=0$，

分别求三次样条插值函数。

### 解答

（1）

给的一阶导数，因此使用三弯矩方程：

$\begin{cases}x_i m_{i-1} + 2m_i + \lambda_i m_{i+1} = d_i \\\lambda_i = \frac{h_i}{h_i + h_{i+1}} \\\mu_i = \frac{h_{i+1}}{h_i + h_{i+1}} \\d_i = 3( \lambda_i \frac{y_{i+1} - y_i}{h_{i+1}} + \mu_i \frac{y_i - y_{i-1}}{h_i} )\end{cases}$

节点：$x_0 < x_1 < \dots < x_n$，步长：$h_i = x_i - x_{i-1}$。

本题为等距，故可简化为

$m_{i-1} + 4m_i + m_{i+1} = 0, i=1,2$，

代入已知解得：

$\begin{cases}m_1 = \frac{4}{15} \\m_2 = \frac{1}{15}\end{cases}$

已知：$S(x) = y_i \frac{(x_{i+1}-x)^3}{6h_i} + y_{i+1} \frac{(x-x_i)^3}{6h_i} + (y_i - \frac{M_i h_i^2}{6})\frac{x_{i+1}-x}{h_i}$

$+ (y_{i+1} - \frac{M_{i+1} h_i^2}{6})\frac{x-x_i}{h_i}$

本题简化为：$S(x) = m_i (-2t^3 + 3t^2) + m_{i+1}(t^3 - t^2)$。

则$x \in [0,1]$时，$t=x$，

$S_0(x) = \frac{19}{15}x^3 - \frac{34}{15}x^2 + x$；

$x \in [1,2]$时，$t=x-1$，

$S_1(x) = -\frac{1}{3}(x-1)^3 + \frac{4}{5}(x-1)^2 - \frac{1}{15}(x-1)$；

$x \in [2,3]$时，$t=x-2$，

$S_2(x) = \frac{1}{15}(x-2)^3 - \frac{2}{15}(x-2)^2 + \frac{1}{15}(x-2)$。

（2）

给的二阶导数，因此使用三弯矩方程：

$\begin{cases}\mu_i M_{i-1} + 2M_i + \lambda_i M_{i+1} = e_i \\\lambda_i = \frac{h_i}{h_i + h_{i+1}} \\\mu_i = \frac{h_{i+1}}{h_i + h_{i+1}} \\e_i = \frac{6}{h_i + h_{i+1}} ( \frac{y_{i+1} - y_i}{h_{i+1}} - \frac{y_i - y_{i-1}}{h_i} )\end{cases}$

节点：$x_0 < x_1 < \dots < x_n$，步长：$h_i = x_i - x_{i-1}$。

本题为等距，故可简化为

$M_{i-1} + 4M_i + M_{i+1} = 0$。

代入已知解得：

$\begin{cases}M_1 = -\frac{4}{15} \\M_2 = \frac{1}{15}\end{cases}$

已知：$S(x) = \frac{M_i}{6h_i} (x_{i+1}-x)^3 + \frac{M_{i+1}}{6h_i} (x-x_i)^3 + (y_i - \frac{M_i h_i^2}{6})\frac{x_{i+1}-x}{h_i}$

$+ (y_{i+1} - \frac{M_{i+1} h_i^2}{6})\frac{x-x_i}{h_i}$

本题简化为：

$S(x) = \frac{M_i}{6} (1-t+x_i)^3 + \frac{M_{i+1}}{6} (x-x_i)^3 - \frac{M_i}{6} (1-t+x_i) - \frac{M_{i+1}}{6} (x-x_i)$，

则$x \in [0,1]$时，$i=0$，

$S_0(x) = \frac{1}{6}(1-x)^3 - \frac{4}{90}x^3 - \frac{1}{6}(1-x) + \frac{4}{90}x$；

$x \in [1,2]$时，$i=1$，

$S_1(x) = -\frac{4}{90}(2-x)^3 + \frac{1}{90}(x-1)^3 + \frac{4}{90}(2-x) - \frac{1}{90}(x-1)$；

$x \in [2,3]$时，$i=2$，

$S_2(x) = \frac{1}{90}(3-x)^3 - \frac{1}{90}(3-x)$。
