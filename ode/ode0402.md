**2**

试求一微分方程，使它有奇解$y=\sin x$。

**解答**

已知Clairaut方程的奇解为

$\begin{cases}x = -f'(p) \\y = -f'(p)p + f(p)\end{cases}$

依据$y = \sin x$，有$f'(p)p - f(p) = \sin(f'(p))$

因此方程

$\begin{cases}y = xp + f(p) \\p = \frac{dy}{dx} \\f'(p)p - f(p) = \sin(f'(p))\end{cases}$

有奇解$y = \sin x$。

**1**

利用线性单摆方程估计读者所在地的重力加速度大小$g$。

**解答**

$m\left(l\frac{d^2x}{dt^2}\right) = -mg\sin x$，

$\frac{d^2x}{dt^2} + \frac{g}{l}\sin x = 0$。

由于$\sin x \sim x$，令$\omega^2 = \frac{g}{l}$，则

$\ddot{x} + \omega^2x = 0$。

令$x = e^{\lambda t}$，代入得$\lambda^2e^{\lambda t} + \omega^2e^{\lambda t} = 0$，

则$\lambda = \pm i\omega$，通解为$x = C_1e^{i\omega t} + C_2e^{-i\omega t}$。

利用Euler公式$e^{i\theta} = \cos\theta + i\sin\theta$，

$x = C\cos(\omega t + \varphi)$。

因此$\omega T = 2\pi \Rightarrow T = 2\pi\sqrt{\frac{l}{g}} \Rightarrow g = \frac{4\pi^2l}{T^2}$。

设$l = 1\mathrm{m}$，测得$T = 2\mathrm{s}$，故$g = 9.87\mathrm{m/s^2}$。

**2**

如果在非线性单摆方程中取$\sin x$的三次近似，即

$\sin x \approx x - \frac{1}{6}x^3$，

则有单摆的三次近似方程

$\frac{\mathrm{d}^2x}{\mathrm{d}t^2} + a\left(x - \frac{1}{6}x^3\right) = 0$。

由此证明单摆振动是不等时的，而且它的相图表明可以发生进动。

**解答**

由方程$\dot{x}x + a\left(x - \frac{9}{x^2}\right) = 0$，$a = \frac{q}{c} > 0$。

两侧同乘$x$：$\dot{x}x = -a\left(x - \frac{9}{x^2}\right)x$。

对$x$积分：$\frac{1}{2}\dot{x}^2 + a\left(\frac{1}{2}x^2 - \frac{9}{2x}\right) = E$（能量守恒）。

由于$\dot{x} = \frac{dx}{dt}$，解得$dt = \frac{dx}{\sqrt{2E - ax^2 + \frac{9}{x}}}$。

$$T = 4\int_0^A \frac{dx}{\sqrt{2E - ax^2 + \frac{9}{x}}}, \ A \text{为最大振幅}.$$

由于$T$中分$A$因，因此不定时。

观察相平面的轨线：

$\frac{1}{2}\dot{x}^2 = E - \frac{a}{2}x^2 + \frac{9}{x}$。

当$E$足够大时，恒$>0$，即摆将持续绕定点旋转，此为进动。
